# Imagens do Site - Guto Spinoza Personal Trainer

## Imagens Necessárias

Para o funcionamento completo do site, você precisará adicionar as seguintes imagens nesta pasta:

### Hero Section
- `hero-trainer.png` - Imagem principal do personal trainer (PNG com fundo transparente, recomendado: 800x1000px)

### Sobre
- `sobre.jpg` - Foto do personal trainer para seção "Sobre" (recomendado: 600x800px)

### Serviços
- `servico-presencial.jpg` - Imagem para treino presencial (recomendado: 500x300px)
- `servico-online.jpg` - Imagem para consultoria online (recomendado: 500x300px)
- `servico-grupo.jpg` - Imagem para treino em grupo (recomendado: 500x300px)

### Clube de Vantagens
- `vantagens.jpg` - Imagem para seção de vantagens (recomendado: 600x600px)

### Depoimentos
- `depoimento1.jpg` - Foto do cliente 1 (recomendado: 100x100px, circular)
- `depoimento2.jpg` - Foto do cliente 2 (recomendado: 100x100px, circular)
- `depoimento3.jpg` - Foto do cliente 3 (recomendado: 100x100px, circular)

### Transformações (Antes e Depois)
- `antes1.jpg` - Foto antes do cliente 1 (recomendado: 300x400px)
- `depois1.jpg` - Foto depois do cliente 1 (recomendado: 300x400px)
- `antes2.jpg` - Foto antes do cliente 2 (recomendado: 300x400px)
- `depois2.jpg` - Foto depois do cliente 2 (recomendado: 300x400px)
- `antes3.jpg` - Foto antes do cliente 3 (recomendado: 300x400px)
- `depois3.jpg` - Foto depois do cliente 3 (recomendado: 300x400px)

## Dicas para as Imagens

1. **Qualidade**: Use imagens de alta qualidade, mas otimizadas para web (comprima antes de usar)
2. **Formato**: Prefira JPG para fotos e PNG para imagens com transparência
3. **Estética**: Mantenha um tom escuro/premium nas fotos para combinar com o design do site
4. **Iluminação**: Fotos com iluminação lateral dourada/quente combinam com a identidade visual

## Ferramentas Recomendadas para Otimização
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (para Mac)
